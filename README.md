# 🍡 Tusok: Streetfood Simulator

A cute little game about being a street food vendor selling fishball and kikiam to hungry children!

## Overview

Tusok is a fun and engaging street food simulator game where you take on the role of a vendor. Manage your shop, serve hungry customers, and build your streetfood empire. This project is currently in **beta testing**.

Built with **JavaFX**, this game combines engaging gameplay mechanics with a charming interface to create an enjoyable casual gaming experience.

## Features

- 🎮 **Casual Gameplay** - Easy-to-learn mechanics perfect for all ages
- 🍴 **Street Food Vendor Simulation** - Sell fishball, kikiam, and more
- 👨‍👩‍👧 **Customer Interactions** - Serve hungry children and build your reputation
- 💻 **JavaFX-based UI** - Clean and intuitive user interface

## Getting Started

### Prerequisites

- Java Development Kit (JDK) 11 or higher
- JavaFX SDK

### Installation

1. Clone the repository:
```bash
git clone https://github.com/iskwipi/Tusok-Streetfood-Simulator.git
cd Tusok-Streetfood-Simulator
```

2. Navigate to the project directory:
```bash
cd tusok
```

3. Compile and run the project with JavaFX:
```bash
javac --module-path /path/to/javafx-sdk/lib --add-modules javafx.controls -d out src/**/*.java
java --module-path /path/to/javafx-sdk/lib --add-modules javafx.controls -cp out tusok.Main
```
